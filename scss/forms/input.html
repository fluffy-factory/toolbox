# Input

# Classic


<label for="text">Input text</label>
<input type="text" id="text" name="text" placeholder="Input text">

$disabled-background: $color-grey;

@include inputPlaceholder(font-style, italic);

input[type="text"],
input[type="email"],
input[type="number"],
input[type="password"],
input[type='datetime'],
input[type="search"],
input[type="url"] {
  @extend .input-reset;
  &:disabled {
    background-color: $disabled-background;
  }
}

# Custom

WARNING

These custom fields require the same HTML structure to work properly.


<div class="input-container">
	<input type="text" id="text" name="text" placeholder="Input text">
	<label for="text">Input text</label>
</div>

$padding-left-right: 20px;
$marg-top-input: 15px;
$font-size-icon: $font-size-base;

$background-label: $color-primary;
$background-disabled-label: $color-grey;
$color-label: white;

@include inputPlaceholder(font-style, italic);

input[type="text"],
input[type="email"],
input[type="number"],
input[type="password"],
input[type='datetime'],
input[type="search"],
input[type="url"] {
  @extend .input-reset;
  &:disabled {
    background-color: $background-disabled-label;

    & + label {
      background-color: darken($background-disabled-label, 50%);
    }
  }
}

.input-container {
  position: relative;

  .#{unquote($font-icon)} {
    position: absolute;
    top: 50%;
    right: $padding-left-right;
    font-size: $font-size-icon;
    transform: translateY(calc(-50% + #{$marg-top-input}/2));

    & + input {
      padding-right: $padding-left-right*2 + $font-size-icon;
    }
  }

  input {
    width: 100%;
    margin-top: $marg-top-input;
    padding: 18px $padding-left-right 15px $padding-left-right;
    background-color: white;
    &:focus + label {
      transform: translateX(5px);
    }
  }

  label {
    position: absolute;
    top: 0;
    left: 0;
    margin: 0 $padding-left-right;
    max-width: calc(100% - #{$padding-left-right});
    padding: 5px 8px;
    color: $color-label;
    background-color: $background-label;
    font-size: $font-size-small;
    line-height: 1;
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
    transition: transform $duration $easeInOutCubic;
  }
}

# Material

WARNING

These custom fields require the same HTML structure to work properly.


<div class="input-validate">
	<i class="icon icon-lock"></i>
	<input type="text" id="text" name="text">
	<label for="text">Input text</label>
</div>

$padding-left-right: 20px;
$padding-bottom: 10px;
$padding-top: 30px;

$background-disabled-label: $color-grey;

@include inputPlaceholder(font-style, italic);

input[type="text"],
input[type="email"],
input[type="number"],
input[type="password"],
input[type='datetime'],
input[type="search"],
input[type="url"] {
  @extend .input-reset;
  &:disabled {
    background-color: $background-disabled-label;
  }
}

.input-validate {
  position: relative;

  input {
    width: 100%;
    background-color: white;
    border: 0;
    border-radius: 5px;
    padding: $padding-top $padding-left-right $padding-bottom $padding-left-right;
    font-size: $font-size-small;
    &:focus + label,
    &[value] + label,
    &.is-valid + label {
      top: 0;
      font-size: $font-size-extra-small;
      color: $color-grey-dark;
      transform: translateY($padding-bottom);
    }
  }

  label {
    position: absolute;
    left: $padding-left-right;
    top: 50%;
    margin: 0;
    font-size: $font-size-small;
    font-weight: $font-weight-semi-bold;
    transform: translateY(-50%);
    pointer-events: none;
    transition: all $duration $easeInOutCubic;
  }

  .#{unquote($font-icon)} {
    position: absolute;
    right: $padding-left-right;
    top: 50%;
    transform: translateY(-50%);
    pointer-events: none;

    & ~ input {
      padding-right: $padding-left-right + $font-size-base*2;
    }
  }
}

WARNING

To have the best results, add MaterialInput.js

# View password


TIP

Works with the same design of Material and Custom input

WARNING

To have the best results, add ShowPassword.js

<div class="input-validate">
	<i data-view-password class="icon icon-eye"></i>
	<input type="password" id="password" name="password">
	<label for="password">Input Password</label>
</div>

$padding-icon: 10px;
$font-size-icon: 20px;

.input-validate, .input-container {
  input[type="password"] {
    font: caption;
    font-size: $font-size-small;
  }

  *[data-view-password] {
    cursor: pointer;
    pointer-events: all;
    font-size: $font-size-icon;
    padding: $padding-icon;
    right: $padding-left-right - $padding-icon;
  }
}

.input-container {
  input[type="password"] {
    font-size: inherit;
  }

  *[data-view-password] {
    transform: translateY(calc(-50% + #{$padding-icon}));
  }
}
