# ✅ Checkboxes and radio

  • Checkbox.scss
  • Radio.scss

# Checkbox

<input type="checkbox" id="miamolex" name="cats">
<label for="miamolex">Miamolex</label>

<!-- disabled -->
<input type="checkbox" id="disabled" name="cats" disabled>
<label for="disabled">disabled</label>

// Checkboxes
%checkbox-icon-initial {
  content: '•';
}

// Use with a font icon 
%checkbox-icon-checked {
  @extend %icon-check;
  font-family: $font-icon;
}

%checkbox-icon-disabled {
  content: '•';
}

// Hide the input
input[type="checkbox"] {
  display: none;
}

# Extend

You can extend the content property to use a custom font.

// Radios
%radio-icon-initial {
  @extend %icon-point;
  font-family: $font-icon;
}

%radio-icon-checked {
  @extend %icon-point;
  font-family: $font-icon;
}

%radio-icon-disabled {
  @extend %icon-point;
  font-family: $font-icon;
}

// Hide the input
input[type="radio"] {
  display: none;
}
